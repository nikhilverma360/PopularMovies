<h2>How to run on your machine?</h2>
<p>The app need themoviedb.org API. You must have your own API key to build the app.</p>
<p>If you do not have a gradle.properties file, create one</p>
<ol>
<li>Go to Project file in the directory</li>
<li>Right click > New > File > gradle.properties</li>
</ol>
<p>Now paste THE_MOVIE_DB_API_KEY="your-api-key-here" in the gradle.properties file</p>
<p>Build the project and Run</p>
